【CSS】修改placeholder的文字颜色

博文 作者：广树 时间：2016-02-15 22:58 分类：CSS/HTML

HTML5提供了一个很便利的功能，就是能然输入框在未输入字符的时候显示提示文字。

如：


但是颜色却无法跟着color走。

这时候就需要用到 


:-moz-placeholder { /* Mozilla Firefox 4 to 18 */
    color: #f00;
    opacity: 1;
}

::-moz-placeholder { /* Mozilla Firefox 19+ */
    color: #f00;
    opacity: 1;
}

input:-ms-input-placeholder {
    color: #f00;
}

input::-webkit-input-placeholder {
    color: #f00;
}
